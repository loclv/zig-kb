# Awesome Zig Ecosystem

This page indexes the [awesome-zig](https://github.com/zigcc/awesome-zig) maintained by ZigCC, covering the breadth of the Zig ecosystem.

## Fundamentals

- Learning Resources: [Zig Language Reference](https://ziglang.org/documentation/master/), [Zig Guide](https://zig.guide), [Ziglings](https://ziglings.org/), [Zen of Zig](https://zenofzig.com)
- Community: [Ziggit](https://ziggit.dev/), [Discord](https://discord.gg/zig), [ZigCC](https://ziglang.cc), [Zig Monthly](https://zig.news/)
- Editors & Tooling: ZLS, VSCode extension, Vim/Neovim plugins, ZigBrains (JetBrains)
- Package & Version Managers: [zigup](https://github.com/marler8997/zigup), [zvm](https://github.com/lispking/zvm), [zigmod](https://github.com/nektro/zigmod)

## Language Essentials

- Memory Allocators: [zimalloc](https://github.com/dweiller/zimalloc), [zigrc](https://github.com/Aandreba/zigrc), [zig-gc](https://github.com/Hejsil/zig-gc)
- Data Structures: [zig-graph](https://github.com/mitchellh/zig-graph), [zig-containers](https://github.com/Sahnvour/zig-containers), [zort](https://github.com/alichraghi/zort)
- String & Text: [zg](https://codeberg.org/atman/zg) (Unicode), [zig-string](https://github.com/JakubSzark/zig-string)
- Parsing & Formats: [zig-yaml](https://github.com/kubkon/zig-yaml), [zimdjson](https://github.com/ezequielramis/zimdjson), [tomlz](https://github.com/mattyhall/tomlz), [ziglyph](https://github.com/jecolon/ziglyph)
- CLI & Args: [zig-clap](https://github.com/Hejsil/zig-clap), [zig-args](https://github.com/MasterQ32/zig-args), [yazap](https://github.com/PrajwalCH/yazap)

## Systems Programming

- Async Runtimes: [libxev](https://github.com/mitchellh/libxev), [zap](https://github.com/kprotty/zap), [pike](https://github.com/lithdew/pike)
- Embedded: [microzig](https://github.com/ZigEmbeddedGroup/microzig), [regz](https://github.com/ZigEmbeddedGroup/regz)
- Operating Systems: [Pluto](https://github.com/ZystemOS/pluto), [birth](https://github.com/davidgm94/birth), [CascadeOS](https://github.com/CascadeOS/CascadeOS)
- Compilers & Interpreters: [arocc](https://github.com/Vexu/arocc), [buzz](https://github.com/buzz-language/buzz), [zua](https://github.com/squeek502/zua)
- Emulators: [zba](https://github.com/paoda/zba) (GBA), [fundude](https://github.com/fengb/fundude) (Gameboy)

## Network & Web

- Build with Zig: Projects using Zig's build system for C/C++ cross-compilation — FFmpeg, SDL, raylib, openssl, and more.
